# Snake Game (C++ / Raylib)

A simple Snake game developed using C++ and Raylib. The game includes classic snake mechanics where the player controls the snake to eat food and grow while avoiding collisions.

## Features
- Real-time movement
- Collision detection (walls and self)
- Score tracking
- Smooth gameplay using Raylib

## Technologies Used
- C++
- Raylib

## How to Run
- Make sure Raylib is installed
- Compile the code using a C++ compiler

Example (Windows with g++):
g++ main.cpp -o snake -lraylib -lopengl32 -lgdi32 -lwinmm

## Controls
- Arrow Keys to move the snake
